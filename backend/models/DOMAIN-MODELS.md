# Domain Model Definitions

This document defines the core entity shapes for the Job Discovery platform. These models map to SQLAlchemy models in MVP 2 and beyond.

## UserProfile
Core settings for job discovery relevance and application generation.
- `id`: UUID (Primary Key)
- `target_roles`: Array of strings
- `preferred_stack`: Array of strings
- `seniority_level`: Enum/String
- `target_salary_min`: Integer (pence)
- `target_salary_max`: Integer (pence)
- `preferred_location`: String
- `notice_period`: String
- `updated_at`: DateTime
- **Constraints/Relationships**: Used for RLS isolation across all user-scoped data.

## CompanyResearch
Intelligence collected on specific companies for interview prep.
- `id`: UUID (Primary Key)
- `company_name_slug`: String (Unique lookup key)
- `research_data`: JSONB
- `sentiment_score`: Float (Optional)
- `funding_stage`: String (Optional)
- `tech_stack`: Array of strings (Optional)
- `culture_signals`: Text (Optional)
- `updated_at`: DateTime

## Job
The core job listing entity.
- `id`: UUID (Primary Key)
- `title`: String
- `company`: String
- `company_slug`: String
- `company_name_slug`: String (For interview prep company research lookup)
- `location`: String
- `source`: String (e.g. 'linkedin', 'jobserve')
- `url`: String (Unique)
- `description`: Text
- `salary_min`: Integer (pence)
- `salary_max`: Integer (pence)
- `currency`: String (default 'GBP')
- `recruiter_id`: UUID (Optional foreign key linking to Recruiter)
- `saved`: Boolean (Computed field populated by left join against SavedJob)
- `scraped_at`: DateTime

## SavedJob
Junction tracking jobs bookmarked by the user.
- `user_id`: UUID (Foreign Key)
- `job_id`: UUID (Foreign Key)
- `saved_at`: DateTime
- **Constraints**: Unique constraint on `(user_id, job_id)`.

## Recruiter
Contact profiles discovered during scraping or manual entry.
- `id`: UUID (Primary Key)
- `name`: String
- `company`: String
- `email`: String (Optional)
- `linkedin_url`: String (Unique dedup key)
- `interaction_score`: Integer
- `notes`: Text
- `created_at`: DateTime

## Application
Tracks the lifecycle of a job application.
- `id`: UUID (Primary Key)
- `job_id`: UUID (Foreign Key)
- `user_id`: UUID (Foreign Key, for RLS isolation)
- `status`: Enum (draft, applied, awaiting_response, interviewing, offered, rejected, withdrawn)
- `notes`: Text
- `applied_at`: DateTime
- `created_at`: DateTime
- `updated_at`: DateTime

## CV
Candidate's resume, parsed and embedded for RAG.
- `id`: UUID (Primary Key)
- `user_id`: UUID (Foreign Key)
- `filename`: String
- `s3_path`: String (Optional)
- `embedding_status`: Enum (pending, processing, ready, failed)
- `parsed_text`: Text (Optional)
- `created_at`: DateTime

## CoverLetter
Generated cover letters bound to a specific job.
- `id`: UUID (Primary Key)
- `job_id`: UUID (Foreign Key)
- `user_id`: UUID (Mandatory for RLS isolation)
- `content`: Text
- `ats_score`: Integer
- `status`: Enum (draft, final, generating, ready, failed)
- `created_at`: DateTime

## ScrapeRun
Audit log for scraping operations.
- `id`: UUID (Primary Key)
- `agent_id`: String (Optional)
- `source_id`: String
- `status`: String (Optional)
- `jobs_inserted`: Integer
- `errors`: Integer
- `duration_seconds`: Float
- `ran_at`: DateTime

## InteractionEvent
Logs interactions between users and recruiters or jobs.
- `id`: UUID (Primary Key)
- `user_id`: UUID (Foreign Key)
- `recruiter_id`: UUID (Optional Foreign Key)
- `job_id`: UUID (Optional Foreign Key)
- `event_type`: String
- `notes`: Text
- `created_at`: DateTime

## interview_questions
Bank of interview questions generated by agents.
- `id`: UUID (Primary Key)
- `company_name_slug`: String (Foreign Key)
- `job_id`: UUID (Optional Foreign Key)
- `question`: Text
- `difficulty_rating`: Integer (1-5 rating of expected difficulty)
- `suggested_answer_points`: Array of strings
- `created_at`: DateTime

## eval_metrics
Storage for evaluation metrics gathered during pipeline execution (e.g., RAG precision).
- `id`: UUID (Primary Key)
- `agent_id`: String
- `metric_name`: String
- `metric_value`: Float
- `context_id`: String
- `created_at`: DateTime

## cv_embeddings
Vector storage for CV chunks via pgvector.
- `id`: UUID (Primary Key)
- `cv_id`: UUID (Foreign Key)
- `chunk_text`: Text
- `embedding`: Vector
- `created_at`: DateTime

## Constraints & Relationships
- **Unique Constraints**:
  - `Job.url`
  - `Recruiter.linkedin_url`
  - `CompanyResearch.company_name_slug`
  - `SavedJob(user_id, job_id)`
- **Foreign Keys**:
  - `CoverLetter.user_id` -> `UserProfile.id` (For RLS isolation)
  - `Application.user_id` -> `UserProfile.id` (For RLS isolation)
  - `CV.user_id` -> `UserProfile.id` (For RLS isolation)
  - `SavedJob.user_id` -> `UserProfile.id` (For RLS isolation)
  - `InteractionEvent.user_id` -> `UserProfile.id` (For RLS isolation)
  - `Job.recruiter_id` -> `Recruiter.id` (For job-to-recruiter navigation)
  - `Job.company_name_slug` -> `CompanyResearch.company_name_slug` (For interview prep company research lookup)
- **Computed Fields**: 
  - `Job.saved` (Left join against `SavedJob`)
