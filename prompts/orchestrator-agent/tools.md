# TOOLS — Orchestrator Agent

This file describes the permitted tools for the Orchestrator Agent.

## Permitted Dispatch Tools
1. **`dispatch_security_check`**: Invoke SecurityAgent safety checks.
2. **`dispatch_ranking_evaluation`**: Call RankingAgent evaluation rules.
3. **`dispatch_context_retrieval`**: Query candidate RAG profile facts.
4. **`dispatch_cover_letter_generation`**: Trigger custom document drafting.
