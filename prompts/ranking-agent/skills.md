# SKILLS — Ranking Agent

This file describes the specialized heuristic capabilities available to the Ranking Agent.

## 1. Skill Extraction
- Detect matching programming languages, libraries, databases, and architectural methodologies in text blocks.
- Perform semantic mapping (e.g. recognize that "PostgreSQL" and "postgres" represent the same tool).

## 2. Seniority Scoring
- Adjust score weight based on matched level indicators (e.g. Senior, Lead, Architect).
- Demote score when experience years or prerequisites don't align with the candidate profile.
