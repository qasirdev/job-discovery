# SKILLS — RAG Agent

This file describes the specialized heuristic capabilities available to the RAG Agent.

## 1. Context Query Generation
- Extract search criteria and keyword weights from job postings.
- Expand acronyms (e.g. mapping "SQL" to "PostgreSQL" or "RAG" to "Retrieval Augmented Generation").

## 2. Experience Relevance Mapping
- Rank candidate projects by tech stack overlapping degree.
- Summarize long historical achievements into short, job-relevant bullet points.
